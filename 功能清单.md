# 功能清单 · 无人车 Web 地面站

> 范围：**仅适配 ArduPilot（APM）固件**，**仅面向无人车（Rover / 地面车）**。
> 不做无人机功能、不做调试/调参/校准功能。本清单为内部维护的开发与验收基线。
> 图例：✅ 已实现并自测　🚧 进行中　⬜ 计划中（按优先级）　❌ 明确不做

最后更新：2026-06-25

---

## 0. 架构 / 基础设施
| 状态 | 项 | 说明 |
|:--:|---|---|
| ✅ | Node 桥接进程 | 持有飞控链路，MAVLink ↔ WebSocket(JSON)；浏览器无法直连串口/UDP，桥接为必需 |
| ✅ | MAVLink 编解码 | 使用 `node-mavlink`（common + ardupilotmega，MAVLink v2） |
| ✅ | 静态 Web 服务 | 桥接进程同端口提供前端页面 |
| ✅ | 自测 / 集成测试 | `selftest.js`（API+编解码闭环）、`itest.js`（假飞控端到端，13/13 通过） |
| ✅ | 仿真环境 + 场景测试 | `sim/rover-sim.js`（行为级 ArduRover 仿真）、`sim/simtest.js`（解锁→围栏→任务驾驶→越界→手柄遥控全链路，8/8 通过） |
| ⬜ | 桥接打包为 Windows 服务/安装包 | 含自启动、COM 口热插拔、代码签名（避免 SmartScreen 拦截） |
| ⬜ | 多车/多链路 | 当前单车单链路；如需车队再扩展 |

## 1. 连接 / 链路
| 状态 | 项 | 说明 |
|:--:|---|---|
| ✅ | 串口 / USB 数传 | 串口号 + 波特率（57600/115200） |

| ✅ | GCS 心跳保活 | 1Hz，向飞控声明 GCS 在线 |
| ✅ | 自动请求数据流 | REQUEST_DATA_STREAM ALL @4Hz |
| ✅ | 断线检测 / 状态提示 | 心跳超时提示；前端自动重连桥接 |
| ✅ | 链路质量显示 | RADIO_STATUS RSSI（本端/对端） |
| ⬜ | 断链失效保护提示 | 与飞控 GCS-failsafe 联动的明确告警 |
